# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript**.
This project demonstrates basic game logic, DOM manipulation, and responsive design.

## 🚀 Features

* 🟢 Two-player mode (X vs O)
* 🔁 Restart game functionality
* 🏆 Winner detection
* 🤝 Draw detection
* 📱 Responsive design (works on mobile & desktop)
* 🎨 Clean and modern UI with gradient background

---
## 🧠 How It Works

* The game uses a **JavaScript array** to track the board state.
* Players take turns clicking on cells.
* After each move:

  * The game checks for a **winner**
  * If no winner, it checks for a **draw**
* Game stops once a result is found.
